Forecast configuration

Forecasts are provided thanks to open-meteo! Here you can configure the location for which you want to receive forecasts. The location can be given by coordinates or by a postal code.

If you want to enter a location using a zip code, you need to get an API key for a OpenWeather geocoding service. The API key can be created after logging in to openweathermap.org. The free plan includes the geocoding service.

The API key should be 32 characters long.
Longer period improves battery life.
The coordinates use the point "." as the decimal separator.
